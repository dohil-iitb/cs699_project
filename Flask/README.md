## Drug Price Tracker
A simple web application to compare drug prices across different pharmacies.
Features
 Search Drug Prices - Find current prices for medications

 Price Comparison - Compare prices across multiple pharmacies

 Price History - View price trends over time

 Pharmacy Comparison - See all drugs and their best prices


# Run the application

python3 app.py

# Open your browser

Go to: http://localhost:5000

# Start searching

Try searching for: aspirin, metformin, or ibuprofen

# How to Use
Search for Drug Prices
Go to the home page

Enter a drug name (e.g., "aspirin")

Click "Search Prices"

View prices from different pharmacies

Compare All Drugs
Click "Compare All Drugs" to see prices for all medications

View Price History
Click "Price History" to see how prices change over time

# Sample Drugs Included
Aspirin

Metformin

Ibuprofen
